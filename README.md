# README

`@umijs/max` 模板项目，更多功能参考 [Umi Max 简介](https://umijs.org/docs/max/introduce)

## 启动项目（start project）

Start the project first:
  ```
    yarn start
    ||
    npm run start
  ```

After the electron frame：
  ```
    yarn start:main
    ||
    npm run start:main
  ```
